**Objeto Virtual de Aprendizaje — Teoría General de Sistemas**  
Universidad Simón Bolívar · Ingeniería de Sistemas · 2025

---

## 📋 Descripción

Objeto Virtual de Aprendizaje (OVA) desarrollado como proyecto final de la asignatura **Teoría General de Sistemas**. La OVA aborda los fundamentos, modelos y aplicaciones de la **Teoría de Colas** y la **Teoría de Juegos**, dos pilares fundamentales del estudio de sistemas complejos.

La OVA está construida como una página web interactiva en HTML5, CSS3 y JavaScript puro, sin dependencias externas ni frameworks. Es completamente funcional desde el navegador.

## 🎯 Objetivos

### General
Diseñar e implementar una OVA funcional, accesible y didáctica que permita comprender y enseñar los fundamentos de la Teoría de Colas y la Teoría de Juegos como ramas de la Teoría General de Sistemas.

### Específicos
- Investigar y sintetizar el marco conceptual de ambas teorías a partir de fuentes académicas en español.
- Aplicar principios de diseño instruccional y experiencia de usuario con navegación responsive.
- Desarrollar contenidos interactivos (juegos y quiz) que fomenten el aprendizaje activo.
- Integrar videos académicos en español como recursos de apoyo audiovisual.
- Publicar la OVA en una plataforma web pública con acceso libre.

## 🚀 Cómo ejecutar localmente

1. Clona o descarga este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/ova-teoria-colas-juegos.git
   ```
2. Abre el archivo `index.html` directamente en cualquier navegador moderno.
3. No requiere servidor, instalación ni dependencias.

> ⚠️ Los videos (miniaturas de YouTube) requieren conexión a internet para cargar las imágenes. Al hacer clic abren YouTube en una nueva pestaña.

## 📤 Publicación

Para publicar la OVA en línea de forma gratuita, elige una de estas opciones:

### GitHub Pages
1. Sube el repositorio a GitHub.
2. Ve a **Settings → Pages**.
3. En "Source" selecciona la rama `main` y la carpeta `/ (root)`.
4. GitHub generará automáticamente el enlace público.

### Netlify
1. Arrastra la carpeta del proyecto a [netlify.com/drop](https://app.netlify.com/drop).
2. Netlify genera el enlace en segundos.

### Vercel
1. Instala Vercel CLI: `npm i -g vercel`
2. Ejecuta `vercel` dentro de la carpeta del proyecto.
3. Sigue las instrucciones para obtener el enlace público.
